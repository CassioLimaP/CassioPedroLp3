Sistema de Venda de Passagens Aéreas é um site desenvolvido com o principal objetivo vender passagens aéreas da maneira mais simples possível. 
A plataforma desenvolvida permite que o usuário escolha, de maneira intuitiva e eficiente, quais voos deseja agendar.

 
A aplicação foi desenvolvida usando o Java como linguagem principal, utilizando como ferramenta de criação das interfaces gráficas o SceneBuilder, 
além da ferramenta MySQL para a criação e gestão do banco de dados.
